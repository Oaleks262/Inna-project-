# SPEC.md — Сайт Інни Ляховської (Корсети, Львів)

## КОНЦЕПЦІЯ

Комерційний сайт майстра корсетного пошиву + Telegram бот для прийому замовлень.
Основна сторінка вже готова (HTML/CSS/JS демо).
Завдання: додати бекенд, портфоліо сторінку, адмінку і Telegram бота.

-----

## СТЕК

- Frontend: HTML + CSS + JS (vanilla, без фреймворків)
- Backend: Node.js + Express
- Зберігання: JSON файли (без БД)
- Telegram: node-telegram-bot-api
- Зображення: Cloudinary
- Auth адмінки: JWT
- Деплой: VPS + PM2 + Nginx

## API ROUTES

### Публічні (без авторизації):

```
GET  /api/portfolio                       — всі видимі роботи
GET  /api/portfolio/:slug                 — одна робота за slug
GET  /api/portfolio?category=wedding      — фільтр за категорією
GET  /api/portfolio?page=1&limit=9        — пагінація
POST /api/requests                        — нова заявка → Telegram
```

### Адмін (JWT required):

```
POST   /api/admin/login
GET    /api/admin/portfolio
POST   /api/admin/portfolio
PUT    /api/admin/portfolio/:id
DELETE /api/admin/portfolio/:id
PUT    /api/admin/portfolio/:id/toggle    — перемикач видимості
GET    /api/admin/requests
PUT    /api/admin/requests/:id/status
POST   /api/admin/upload
GET    /api/admin/settings
PUT    /api/admin/settings
```

### Режим 1 — Нотифікатор (заявки з сайту)

При POST /api/requests бот миттєво надсилає Інні:

```
🆕 Нова заявка з сайту

👤 Ім'я: {name}
📱 Телефон: {phone}
📐 Розмір: {size}
👗 Робота: {portfolioItem}
💬 Побажання: {message}
📍 Джерело: {source}
🕐 {createdAt}
```

Inline кнопка: [✓ Позначити як переглянуто]
При натисканні → статус заявки в requests.json міняється на “viewed”
→ бот відповідає: “✓ Позначено як переглянуто”

## ФАЗИ (одна фаза → зупинись → чекай підтвердження)

### Фаза 1 — Бекенд

Express сервер + всі API routes + JSON читання/запис +
JWT middleware + Cloudinary upload + multer.

Результат перевірки:

```
GET  /api/portfolio       → повертає масив
POST /api/requests        → зберігає в JSON
POST /api/admin/login     → повертає JWT токен
POST /api/admin/portfolio → додає роботу (з токеном)
```
